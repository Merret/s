# Merret Shorten URL Service

**An Free and Powerful Shorten URL Service**

## How To Use

1. Go to [go-to.ml](https://go-to.ml/console).

2. Paste long URL in the input field.

3. Press "Shorten it!" button.

4. Enjoy your short URL!
